The Yearly Yearbook

➡️ What is this?

This is the Yearly Yearbook, a page where you can add your own custom quotes to the list. The Yearly Yearbook's current state will be archived every year. Archives will be stored in the folder named "Yearly Yearbook Past Editions".

➡️ How can I add a quote?

You can add a quote by filling out the form below, and clicking Submit. Try to only add one quote per year.
